Un programa hecho en java para familiarizarme más con el lenguaje y también con la libreria integrada swing.
En este programa se pueden crear tareas y luego, dentro de cada tarea se pueden agregar pasos. Cada paso puede marcarse como completo,
y una vez que todos los pasos de una tarea están completos, la tarea es automáticamente marcada como completada también.
Las tareas o pasos completadas aparecen como tachadas, y la fecha de finalización aparece a su derecha.

Pude practicar varias cosas durante el proyecto, como por ejemplo a extender clases nativas de java, como por ejemplo varios componentes
de la libreria swing, como JButton o JFrame. También a implementar algunas interfaces existentes como ActionListener.
También descubri los *Look and Feel*, donde modifique el existente nimbus guiandome con la documentación en Oracle.
Otra cosa que aprendí fue que los JLabel pueden renderizar html escrito en ellos, con estilos de css y todo.
